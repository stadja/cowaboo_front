Diigo

the username of whose diigo library you want to access

The password of the user (it won't be saved, but only used in order to generate the basic authentication key that is asked by Diigo to connect to the api)

Your generated authentication key

Zotero

The zotero library you want to access group's id. To find it, it is a bit tricky:

  1. Go to your zotero's group list
  2. Then go to your group library (click, "Group Library")
  3. In the bottom left, you'll find "Subscribe to this feed"
  4. Copy this link, it shoud be something like 
    https://api.zotero.org/groups/303941/items/top?start=0&limit=25&format=atom&v=1
  5. Take the number between "groups/" and "/items": It is your Zotero group id

Your personal zotero Api key. It is a key that identify the bearer and grant permission to write or read the user or group library. This key can be created or found in your zotero's settings (it is called "private key").
